Desenvolvimento web

O que e desenvolvimento web?

o desenvolvimento Web inclui criação, codificação e programação de sites e seu respectivos elementos e linguagem. Ele também envolve integração com sistemas e ferramentas, armazenamento de banco de dados, cibersegurança, entre outros, com o objetivo de estruturar cada detalhe de uma página da internet.

O que faz desenvolvedor web

O desenvolvedor de sites, chamado de profissional de Dev ou web developer, é o responsavel por criar todo os planejamento estrutural da páginas, ultilizando linguaens de programação como HTML, javaScript,PHP, entre várias outras.
